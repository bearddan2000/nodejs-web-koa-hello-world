# nodejs-web-koa-hello-world

## Description
A POC for Koa in nodejs.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
